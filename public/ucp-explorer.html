🤖 Agent Discovery Flow — What Happens Step by Step
1

Agent hits a paywall

The agent calls GET /api/premium-data and receives HTTP 402 Payment Required with a WWW-Authenticate header pointing to the UCP manifest.

2

Agent reads the manifest

The agent fetches GET /.well-known/ucp and reads the business profile, service endpoint, official checkout capability, and TRON payment handler metadata.

3

Agent creates a checkout session

Using the business profile, the agent calls POST /ucp/v1/checkout-sessions with the requested line items. Your gateway freezes the order until you approve via Telegram.

4

You approve on Telegram

A push notification arrives on your phone. Tap Approve and the gateway releases the payment challenge to the agent.

5

Agent pays on-chain

The agent signs and broadcasts a TRC20 USDT transfer to your merchant address. Private keys never leave the agent's machine.

✓

Access granted

Your gateway verifies the transaction on the TRON blockchain and grants the agent access to the premium resource.
